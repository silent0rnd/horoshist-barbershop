# Хорошист - барбершоп

Статический сайт барбершопа «Хорошист» в Ростове-на-Дону.

## Запуск на компьютере

1. Установите Node.js версии 20 или новее.
2. В папке сайта выполните `npm install`.
3. Запустите `npm run dev`.
4. Откройте адрес, который появится в окне запуска - обычно `http://localhost:3000`.

## Проверка сборки

```bash
npm run lint
npm run build
```

После сборки готовая версия лежит в папке `dist`.
